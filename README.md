# WBS_Excel

A Work Breakdown Structure (WBS) automation tool built with Microsoft Excel.
This repository contains an Excel file that helps project managers quickly create and manage WBS documents.

## Overview

`人事業務BPOベンダー変更プロジェクト WBS V0.0.xlsx` is a template Excel file designed to streamline the creation of WBS for HR BPO (Business Process Outsourcing) vendor change projects. It provides a structured format for defining tasks, assigning responsibilities, and tracking project progress.

## Features

- Pre-built WBS structure for HR BPO vendor transition projects
- Task hierarchy with clear breakdown of deliverables
- Easy-to-use Excel format — no special software required
- Customizable for different project scopes and timelines

## Requirements

- Microsoft Excel 2016 or later (or compatible spreadsheet software)

## Usage

1. Download the `.xlsx` file from this repository.
2. Open it in Microsoft Excel.
3. Customize the project name, tasks, owners, and dates to match your project.
4. Use the built-in structure as a starting point to expand your WBS.

# WBS_Excel（日本語）

Microsoft Excel を使った WBS（Work Breakdown Structure / 作業分解構造）自動作成ツールです。
プロジェクトマネージャーが効率的に WBS を作成・管理できる Excel ファイルを提供します。

## 概要

`人事業務BPOベンダー変更プロジェクト WBS V0.0.xlsx` は、人事業務の BPO ベンダー変更プロジェクトにおける WBS 作成を効率化するための Excel テンプレートです。タスクの定義、担当者のアサイン、進捗管理に必要な構造をあらかじめ用意しています。

## 主な機能

- 人事 BPO ベンダー移行プロジェクト向けの WBS 構造をあらかじめ用意
- 成果物を明確に分解したタスク階層
- 特別なソフトウェア不要の使いやすい Excel 形式
- プロジェクトの規模やスケジュールに合わせてカスタマイズ可能

## 動作環境

- Microsoft Excel 2016 以降（または互換性のあるスプレッドシートソフトウェア）

## 使い方

1. このリポジトリから `.xlsx` ファイルをダウンロードします。
2. Microsoft Excel で開きます。
3. プロジェクト名・タスク・担当者・日付を自分のプロジェクトに合わせて編集します。
4. あらかじめ用意された構造を出発点として、WBS を展開・拡張してください。
