## WBS_Excel

A Work Breakdown Structure (WBS) template/tool built with Microsoft Excel.
This repository provides Excel files that help project managers quickly create and manage WBS documents for **any kind of project**.

### Overview

- **WBS_Excel_V0.1.xlsx** is the **main** WBS Excel template.
- **WBS_Excel_V0.1_JP Example.xlsx** is a **Japanese example** version.

The sample content included in the sheets (e.g., an HR BPO vendor change project) is provided **only as a usage example**.
The main purpose of this repository is that you can create and manage a WBS in Excel regardless of what the project is.

### Features

- Create and manage a WBS in a familiar Excel format — no special software required
- Task hierarchy with clear breakdown of deliverables
- Customizable for different project scopes and timelines
- Example content included in the sheets (can be replaced with your own project)

### Requirements

- Microsoft Excel 2016 or later (or compatible spreadsheet software)

### Usage

1. Download the `.xlsx` file(s) from this repository.
2. Open in Microsoft Excel.
3. Replace the example project information with your own project name, tasks, owners, and dates.
4. Use the structure as a starting point to expand your WBS.

## WBS_Excel（日本語）

Microsoft Excel を使った WBS（Work Breakdown Structure / 作業分解構造）テンプレート／作成ツールです。
プロジェクトの種類を問わず、WBS を Excel で作成・管理できるファイルを提供します。

### 概要

- **WBS_Excel_V0.1.xlsx**：メインの WBS Excel テンプレートです。
- **WBS_Excel_V0.1_JP Example.xlsx**：日本語版の使用例（サンプル）です。

シート内に含まれる「人事業務BPOベンダー変更プロジェクト」などの内容は、あくまで **入力例（使用例）** として掲載しているものです。
本リポジトリの主目的は、特定のプロジェクトに依存せず、どのようなプロジェクトでも WBS を Excel で作成できることです。

### 主な機能

- 特別なソフトウェア不要の使いやすい Excel 形式
- 成果物を明確に分解したタスク階層
- プロジェクトの規模やスケジュールに合わせてカスタマイズ可能
- シート内に入力例を収録（自分のプロジェクトに置き換えて利用可能）

### 動作環境

- Microsoft Excel 2016 以降（または互換性のあるスプレッドシートソフトウェア）

### 使い方

1. このリポジトリから `.xlsx` ファイルをダウンロードします。
2. Microsoft Excel で開きます。
3. 使用例のプロジェクト情報を、自分のプロジェクト名・タスク・担当者・日付に置き換えます。
4. あらかじめ用意された構造を出発点として、WBS を展開・拡張してください。
